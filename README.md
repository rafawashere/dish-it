# 🍽️ Dish It!

**Version 0.01**

Dish It! is a simple and intuitive web application designed to help you organize your dinners and never wonder "what's for dinner" again. By maintaining your personal list of favorite meals, you can automatically generate a structured dinner plan for any number of days, governed by custom constraints like a minimum spacing interval to prevent consecutive repeats.

## Key Features

- **Automated Meal Planning:** Instantly generate a dinner plan for 5 days, 7 days, or a custom duration.
- **Meal Library Management:** Easily manage your roster of meals.
- **Repeat Control:** Set a "No Repeat Days" buffer to ensure variety in your dinner rotations.
- **Data Portability:** Move your saved meal lists between devices using the JSON export and import tools.
- **Offline First:** Keep your data private with local storage persistence.

## Getting Started

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```
